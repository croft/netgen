LB-Jobs Tasks

Tasks to submit job implementations to the lb-jobs service, and then create new jobs, wait for their termination and collect the results.

Upload a job implementation.

Inputs:

@param job_impl_id Name of the job implementation (needs to be unique)

@param archive_url S3 URL of the .tar.gz archive of the implementation

@param metadata key=value pairs of metadata on the job implementation

@param service service URL

@param username username used for authentication

@param key private key used for authentication

Outputs:

@return job_id Uploads are asynchronously processed and the progress should be checked with this id.

@return job_impl_id If input id was not given, then the unique id chosen. Otherwise, the input id.

task lb.jobs.UploadJobImpl(
    job_impl_id?,
    archive_url,
    metadata*?,
    service = "https://steve.logicblox.com/job",
    username,
    key,
    driver_meta?
  ): (job_id, out_job_impl_id?) extends AsyncTask(
    driver_meta = $driver_meta,
    task_implementation_class = "com.logicblox.workflow.task.steve.UploadJobImpl"
  )

@param service service URL

@param username username used for authentication

@param key private key used for authentication
