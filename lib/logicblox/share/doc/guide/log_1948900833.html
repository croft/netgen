Log Tasks

Tasks related to logging workflow activity, both in the internal log as well as using the notification mechanisms for external tools, such as Amazon SNS.

Create an event and submit it using notifiers.

The event is created according to the Event message protocol (see task_config.proto) and sent to all configured notifiers. The message optionally contains information about this task, such as id and workspace (see the TaskEvent message).

@param event a string describing the event. This will be set on the required event field of the Event message.

@param notifiers the names of notifier sections in lb-workflow-driver.config that configure notification mechanisms to deliver the event.

@param details optional details to add to the corresponding message field.

@param add_task_info if true, include a TaskEvent field in the Event message that identifies this task.

task lb.log.Notify(
    event,
    notifiers*,
    details?,
    add_task_info = true,
    driver_meta?
  ) extends AsyncTask(
    driver_meta = $driver_meta,
    task_implementation_class = "com.logicblox.workflow.task.log.Notify"
  )
